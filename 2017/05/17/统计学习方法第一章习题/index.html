《统计学习方法》第一章习题

发表于 2017-05-17

1.1

三要素：模型，策略，算法

模型：伯努利模型是定义在取值为0与1的随机变量上的概率分布
策略：对数损失函数 L(Y,P(Y|X)) = -logP(Y|X)
算法：极大似然估计所使用的算法是求取经验风险函数的极小值，贝叶斯估计所使用的算法是求取参数的后验分布，然后计算其期望（贝叶斯不太懂）

为了得到使经验风险最小的参数值，使用的算法都是对经验风险求导，使导数为0

极大似然估计

经典统计学派：模型已定，参数未知，参数是固定的，未知
P(A=1)=θ,P(A=0)=1−θ
独立抽取n个同分布的随机变量A1,A2,…,An。使用极大似然估计即求取以下经验风险函数的极值点
L(P)=−∑(logP(Ai)) = −klogθ−(n−k)log(1−θ)
求导：
θ=k/n

贝叶斯估计

贝叶斯统计学派：通过观察到的现象对概率分布中的主观认定不断进行修正。
(不懂，待下次)

1.2
模型是条件概率分布：P(Y|X)，
损失函数是对数损失函数：L(Y,P(Y|X))=−logP(Y|X)，
经验风险为：
Remp(f)=1/N∑(L(yi,f(xi)))
=1/N∑-logP(yi|xi)
=-1/N*∑logP(yi|xi)
最小（负的log和）=最大（log和）=最大（log内部积）
